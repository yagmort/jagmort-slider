jagmort-slider
==============

Basic image slider plugin for jQuery.

Usage
-----

In most cases only one option you need is duration:

	$("#your-selector").jagmortSlider({
		"duration": 2000
	});

Available options
-----------------

duration
	in msec, default is 4000

showSlideSelector
	default is ".show"

showSlideClass
	default is "show"
